# 关卡设计 AI 协作与自动化工作流 (AI_PROMPT_WORKFLOW)

**核心法则**：把 AI 当成**"关卡主策划兼搜集员"**，把 Unity Test Console 当成**"关卡打印机"**。
**防重铁律 (Artifact-based State Passing)**：无记忆的 AI + 携带历史指纹的本地实体文档 = 彻底治愈 AI 失忆症，实现跨账号无限防重循环。
**排版铁律**：永远不要在 Scene 视图里痛苦地拖拽物体对齐网格，**文本框里的 ASCII 字符是你唯一的真相源 (Source of Truth)**！

本指南提供两套极速关卡生成流水线，请根据场景灵活切换：
- **【路线 A：全自动流水线】**：利用本地工程 AI（Manus / Cursor / Windsurf），让其全网自动搜集、自动排版、自动写档防重。（**终极主战流派**）
- **【路线 B：半自动视觉流水线】**：针对你在网上看到的特定"神仙截图"，让网页大模型看图精准拆解。

---

## 全景流水线总览

```
路线A: 开新对话 → AI 读防重库 → 全网搜集灵感 → 转译 ASCII → 写入模板 → 更新登记簿 → Git Push
路线B: 截图 → 复制防重摘要 → 发给网页 AI → 拿到 ASCII → 手动更新登记簿
共同: 粘贴进 Unity 大文本框 → 1秒微调 → 注入灵魂 → 一键换肤 → 沉淀片段
```

---

## 路线 A：全自动流水线（本地 Agent 自动搜网与扩容）

**适用工具**：Manus, Cursor, Windsurf 等具备【本地文件读写、联网搜索、Git 权限】的 AI 智能体。
**使用场景**：你想挂机扩充关卡库。不管你是换了新电脑、新账号还是新对话窗口，直接把这段话扔给 AI，它会自己去网上找新灵感、写关卡、**并且自动更新防重登记簿**，完美实现无脑良性循环。

### 复制发送以下指令（开新对话时，附带 GitHub Token 直接发送）：

```text
【关卡全网自动搜集、转化与防重归档闭环任务】

GitHub Token: [你的Token]
仓库：https://github.com/jiaxuGOGOGO/MarioTrickster

本次任务：你现在是《MarioTrickster》的主关卡策划。请利用联网搜索，帮我全网搜集 2 个全新的 2D 平台跳跃关卡灵感，并转化为我的白盒 ASCII 模板。

搜集标准（极度重要）：
- 不止搜集极难毒图！有趣的、精妙的、创意独特的关卡设计同样欢迎
- 优先搜集我的字典中【已有机制的新颖组合】，其次搜集【需要新机制的灵感】
- 关注维度：惯性诱导、极限视野盲区、多重弹射链、强制节奏切换、路径分支、追逐压力、环境交互、垂直下落、敌人利用、渐进揭示等

为确保跨会话、跨账号的绝对"零重复"，请严格按以下自动化流水线静默执行，无需向我确认，一气呵成：

📥 步骤 1：克隆仓库并强制查阅防重库（读取状态/极度重要）
- 用 Token 克隆仓库。
- 静默读取 `LevelDesign_References/TEMPLATE_REGISTRY.md` 文件。
- 深刻理解我已经拥有的【核心博弈组合】、【已探索灵感来源】和【尚未覆盖的博弈维度】。
- 警告：你接下来的搜网来源和设计的博弈心理，【绝对禁止】与登记簿中已有内容雷同！

🌐 步骤 2：定向狩猎与变异搜集（避开黑名单）
- 动用搜索查找 2D 平台游戏的优秀关卡设计（搜索范围包括但不限于）：
  · 高难度：马里奥制造2极难毒图、蔚蓝速通路线、I Wanna系列、空洞骑士白宫
  · 创意型：Baba Is You 空间利用、Fez 视角切换、VVVVVV 重力翻转
  · 经典型：洛克人精妙关卡、银河战士探索设计、Ori 弹射组合
  · 休闲有趣：马里奥制造2热门关卡、Rayman 节奏关卡、Shovel Knight 宝藏关
- 必须彻底避开登记簿中【已探索灵感来源】列出的游戏+具体关卡组合。
- 重点提取：局部挑战片段（1~2屏大小），不要整张超大地图。

✍️ 步骤 3：严格转译并写入（生成真相源）
- 物理法则：1字符=1格子。水平最多跨4格，垂直最多跳3格。绝对不可死路。
- 排版死命令：纯空气必须用点号 `.` 严禁空格，保持完美矩形网格（宽20~30，高10~15）。
- 字典（当前已实现）：
  `#`地面, `=`平台, `-`单向平台, `W`墙壁, `^`地刺, `~`火焰, `P`摆锤,
  `B`弹跳平台, `C`崩塌平台, `>`移动平台, `E`弹跳怪, `e`简单敌人,
  `F`伪装墙, `H`隐藏通道, `o`金币, `M`起点, `G`终点。
- 如果灵感中包含字典里没有的机制，用最接近的字符暂代，并在【缺失机制分析】中详细说明。
- 将新关卡 ASCII 矩阵（含动态参数建议、博弈解析、缺失机制分析）追加写入到
  `LevelDesign_References/ASCII_Templates.md` 的末尾，顺延 T 编号。

📝 步骤 4：生成语义指纹并更新登记簿（保存状态记忆/实现闭环）
- 修改 `LevelDesign_References/TEMPLATE_REGISTRY.md` 文件：
  1. 在【完整登记表】追加新行（编号、博弈组合摘要、涉及元素、尺寸、日期、难度标签、灵感来源）。
  2. 在【快速复制区】的代码块中追加对应的摘要行。
  3. 在【已探索灵感来源】中追加本次搜索的游戏名+具体关卡/视频。
  4. 如果新模板覆盖了【灵感池】中的某个博弈维度，标注 [已覆盖: T0x]。
  5. 如果发现了新的缺失机制，追加到【缺失机制待办】中。

🚀 步骤 5：同步云端（持久化记忆）
- 执行 `git add . && git commit -m "feat(Level): 自动搜网新关卡并更新登记簿" && git push`。
- 完成后向我汇报摘要，包括：新模板编号、核心博弈、灵感来源、发现的缺失机制。
- 我去 Unity 中 Pull 后测试。
```

## 路线 B：半自动视觉流水线（网页 AI 看图拆解）

**适用场景**：你在 B 站、YouTube 或 Pinterest 看到一张神仙关卡的截图，想让拥有最强视觉的模型（如网页版 Claude / GPT-4o / Gemini）帮你精准转成 ASCII，且不能和你已有的关卡撞车。

### 第一步：加载外挂记忆

打开本地的 `LevelDesign_References/TEMPLATE_REGISTRY.md`，全选复制**【快速复制区】**的文本。

### 第二步：念动咒语

把局部难点截图发给网页 AI，并粘贴以下提示词（在 `<memory>` 处粘贴刚才复制的黑名单）：

```text
【关卡视觉拆解与防重 ASCII 生成任务】
我上传了一张优秀 2D 平台跳跃关卡的截图。请提取图中最精妙的局部挑战，转化为我的白盒 ASCII 模板。

<memory>
🚫 【绝对去重约束】（核心博弈组合避让）：
我已有以下博弈组合。如果图中的套路我已经有了，请强制加入"追逐逃亡/视野限制/多重时差陷阱"等变异元素进行二次魔改，绝对不可雷同！
[在此处粘贴你刚才复制的 TEMPLATE_REGISTRY.md 快速复制区文本]
</memory>

=== 物理法则与字典映射 ===
1. 排版死命令：空气区域必须用点号 `.` 填充，绝对不能用空格。完美矩形网格（宽20~30，高10~15）。
2. 字典（当前已实现）：
   `#`地面, `=`平台, `-`单向平台, `W`墙壁, `^`地刺, `~`火焰, `P`摆锤,
   `B`弹跳平台, `C`崩塌平台, `>`移动平台, `E`弹跳怪, `e`简单敌人,
   `F`伪装墙, `H`隐藏通道, `o`金币, `M`起点, `G`终点。

=== 严格输出格式要求 ===
请严格按以下 4 个模块输出：
1. 【动态参数建议】：移动平台/陷阱的运动轨迹建议；图中哪些机制我字典里没有？
2. 【ASCII 矩阵代码块】：用 Markdown 代码块包裹，供我一键复制。
3. 【缺失机制分析】：图中有哪些机制我的字典里没有？它带来了什么博弈乐趣？建议用什么字符暂代？
4. 【一键防重登记指纹】：格式严格为 `T编号_[机制A] + [机制B]（核心博弈体验）`。
```

### 第三步：人工闭环操作（极度重要！）

拿到网页 AI 回复后，你**必须**手动完成以下操作：
1. 把生成的 ASCII 矩阵追加到 `LevelDesign_References/ASCII_Templates.md` 末尾。
2. 把【一键防重登记指纹】追加到 `LevelDesign_References/TEMPLATE_REGISTRY.md` 的【完整登记表】和【快速复制区】中。
3. 在【已探索灵感来源】中记录截图来源。
4. `git add . && git commit -m "feat(Level): 视觉拆解新关卡" && git push`。

## 统一实操阶段：Unity 内的极速微操与包浆

不管你用哪条路线拿到了 ASCII 文本，接下来都在 Unity 中执行以下操作：

### 第一阶段：文本拼装与"1秒微调"

1. **一键打印**：在 Unity 按 `Ctrl+T` 唤出 Level Studio。把 AI 生成的字符矩阵粘贴到大文本框中，点击 **"Build From Text"**，关卡骨架瞬间拔地而起。
2. **积木式拼接长关卡**：在大文本框最上方，用右侧 **Snippet Library (片段库)** 追加 `[教学起点区]`；在最下方追加 `[终点冲刺区]`，瞬间拼出完整流程。
3. **"1秒微操"极速迭代（核心精髓）**：
   - 进去测试发现地刺跳不过去？退出 Play 模式，**绝不要去 Scene 里用鼠标拖拽白盒！直接在大文本框里改！**
   - 找到代表地刺的 `^`，选中前面的点号 `.`，按退格键删掉一个点号。地刺瞬间向左平移了一格。
   - 觉得刺太多？把 `^` 改成 `.`，刺就消失。点击 Build 秒更新！_（将调关卡的闭环从 1 分钟缩短到 3 秒钟）_

### 第二阶段：注入灵魂与换位思考

1. **动态参数注入**：对照 AI 给的参数建议，在 Scene 里选中移动平台 `>` 修改位移，选中摆锤 `P` 调整幅度。
2. **Trickster 审查测试**：开启全局作弊的"无冷却/无限能量"，用捣蛋者视角逛一圈。
   - **思考**："如果我是捣蛋者，我要怎么阴人？"
   - **补刀**：如果发现某处对闯关者太安全（纯跳跃没东西附身），就在文本框里打字加一排 `P` 摆锤或 `F` 伪装墙，再次生成。

### 第三阶段：资产沉淀与一键换肤

1. **一键换肤 (Apply Theme)**：白盒测试彻底满意后，去 Test Console 的 **Theme Profile** 槽位拖入配置好的美术包，点击 **Apply Theme**，满屏方块瞬间变成精美完整的游戏画面！`Ctrl+S` 保存场景。
2. **沉淀专属 Snippet**：把微调出手感极其完美的跳跃组合（如极限三连跳）保存下来，下次让代码 AI 直接硬编码进你的 Snippet Library，实现全项目一键复用。

## 隐藏机制：你的"武器库"是如何无限扩充的？

这是关卡设计（文本）与系统开发（代码）完美解耦的超神时刻。

在搜集和拆解过程中，AI 可能会告诉你："某游戏有一条 **传送带** 极其巧妙，但你的字典里没有"。

你不需要在做关卡时去死磕代码，只需要把"传送带"记在 `TEMPLATE_REGISTRY.md` 的【缺失机制待办】里。明天专门写代码时，直接给本地工程 AI 下发这样一个极简任务：

> "本次任务：我需要新增【传送带】机制。请新建 `ConveyorBelt.cs` 继承自 `ControllableLevelElement`。然后在 `AsciiLevelGenerator.cs` 的字典里加两行映射：`<` 代表向左传送，`>` 代表向右传送。写好直接推送。"

代码 AI 像流水线工人一样把底层零件造好。**当你 Pull 代码后，你的"打字密码本"里就永远多了 `<` 和 `>` 两个符号！** 你立刻就能在文本框里敲出传送带，玩法无限裂变扩容！
