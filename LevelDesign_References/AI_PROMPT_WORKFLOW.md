# 关卡设计 AI 视觉拆解工作流（优化版）

> **核心目标**：利用网页端视觉大模型（如 Claude 3.5 Sonnet 或 GPT-4o）将优秀的 2D 平台跳跃关卡截图转化为 Unity Level Studio 可直接使用的 ASCII 白盒模板。
>
> **优化点**：加入了基于"核心博弈组合"的去重机制，避免跨会话/跨账号时 AI 生成雷同的关卡设计。

---

## 完整提示词（咒语）

每次向 AI 发送关卡截图时，请**直接复制粘贴以下全部内容**。

其中第 4 点【去重约束】的内容，请每次从 `TEMPLATE_REGISTRY.md` 的【快速复制区】获取最新版本替换进去。

```text
【关卡视觉拆解与 ASCII 模板生成任务】
我上传了一张优秀 2D 平台跳跃关卡的截图。请提取图中最精妙的局部挑战片段，转化为我的游戏白盒 ASCII 模板。

1. 物理法则与排版限制（极度重要）：
- 1 个字符 = 1x1 格子。我的角色极限：水平最多跨 4 格深渊，垂直最多跳 3 格高台。绝对不可设计出物理死路。
- 每一行的字符数量必须绝对相等，构成完美的矩形网格（宽 20~30，高 10~15）。
- 【排版死命令】：空气/空白区域必须用英文字符点号 `.` 填充，绝对不能用空格（防止排版错乱）。

2. 字典映射（严格使用这些字符，图中有新机制用相近元素暂代）：
`#`地面, `=`平台, `-`单向平台, `W`墙壁, `^`地刺, `~`火焰, `P`摆锤, `B`弹跳平台, `C`崩塌平台, `>`移动平台, `E`弹跳怪, `e`简单敌人, `F`伪装墙, `H`隐藏通道, `o`金币, `M`起点, `G`终点。

3. 附加输出要求：
- 输出 ASCII 矩阵（用代码块包裹，方便我一键复制）。告诉我如何复制不会很多复制到一起去了
- 【动态参数建议】：说明图中的移动平台、陷阱应该是怎样的运动轨迹或速度。
- 【缺失机制分析】：图中有哪些机制我的字典里没有？它带来了什么博弈乐趣？

4. 【去重约束】（核心博弈组合避让）：
我已有以下博弈组合的模板，请避免生成雷同设计（同类型但不同博弈手法的变体欢迎）：
- 静态地刺阵列 + 悬浮诱饵金币 + 安全平台落脚点（被动风险选择）
- Z字交错固定平台 + 平台上巡逻敌人（节奏观察 + 时机起跳）
- 双移动平台(水平+垂直) + 中间弹跳怪衔接（三段连锁跳跃）
- 上下火焰夹道 + 多摆锤错相位（纯时间窗口通过）
- 全崩塌平台路径 + 底部地刺 + 弹跳平台兜底（不停跑的时间压力）
- 伪装墙隐藏密室 + 隐藏通道入口 + 金币奖励（探索发现）
请优先提取图中我【没有】的博弈组合，例如：追逐逃亡、视野限制、路径分支、环境交互、垂直下落、弹射链等。
```

## 工作流闭环指南

### 第一阶段：狩猎灵感与 AI 拆解（在网页浏览器）

1. **寻找切片**：去 B 站搜《马里奥制造2 毒图/精妙关卡》、《蔚蓝 Celeste 速通》或在 Pinterest 搜 "2D platformer level design"。截取**1~2 个屏幕大小的局部难点**（不要截整张超大地图，AI 看不清细节）。
2. **呼叫主策划**：打开网页端视觉能力最强的大模型（推荐 Claude 3.5 Sonnet 或 GPT-4o）。
3. **念动咒语**：把截图发给它，并附上上面的完整提示词。

### 第二阶段：文本拼装与微调（在 Unity Level Studio）

1. **一键打印**：在 Unity 里按 `Ctrl+T` 唤出 Level Studio。把网页 AI 生成的包含 `.` 的字符矩阵复制，粘贴到 `Custom Template Editor` 的大文本框中。点击 **"Build From Text"**，关卡骨架瞬间拔地而起。
2. **积木式拼接长关卡**：如果一个切片太短，利用右侧的 **Snippet Library (片段库)**。在大文本框最上方敲回车，选一个 `[基础平地起点区]` 点击追加；在最下方选一个 `[胜利终点区]` 追加。你瞬间拼出了一个完整的测试流程！
3. **“1秒微操”极速迭代（神级操作）**：
   - 进去跑酷测试，发现有个地刺跳不过去？
   - **退出 Play 模式，不要去 Scene 里用鼠标拖拽白盒！直接在大文本框里改！**
   - 找到那个代表地刺的 `^`，选中它前面代表空气的 `.`，**按退格键删掉一个点号**。地刺瞬间向左平移了一格。
   - 觉得刺太多？把 `^` 改成 `.`，刺就消失了。
   - 点击 Build 生成，秒更新进去接着跳！_（你调整关卡的闭环从 1 分钟缩短到了 3 秒钟）_

### 第三阶段：登记归档（防重复）

1. 当你在 Unity 中微调好一个满意的模板后，打开 `TEMPLATE_REGISTRY.md`。
2. 在【完整登记表】中追加一行，用一句话总结它的**核心博弈组合**（例如："单向平台下落 + 底部巡逻敌人 + 隐藏通道逃生"）。
3. 将这句话同步更新到 `TEMPLATE_REGISTRY.md` 的【快速复制区】。
4. 下次向 AI 发送新截图时，复制最新的【快速复制区】内容替换提示词的第 4 点。
