# MarioTrickster — 关卡模板登记簿

> **用途**：跨会话/跨账号去重。每次让 AI 生成新模板前，复制下方【去重摘要】粘贴到提示词中即可。
>
> **去重粒度**：按"核心博弈组合"而非"类型"去重。同一类型（如地刺深渊）只要博弈手法不同就不算重复，鼓励同类型的深度探索。

## 尚未覆盖的博弈维度（灵感池）

> 以下是经典 2D 平台跳跃中常见但本项目模板尚未触及的博弈维度。搜集新参考时可以优先瞄准这些方向。

| 博弈维度 | 示例手法 | 经典出处 |
|----------|----------|----------|
| **追逐压力** | 不可消灭的追逐者从身后逼近，迫使持续前进 | Celeste (Badeline), Crash Bandicoot (巨石) |
| **视野限制** | 黑暗关卡只有局部光源，或浓雾遮挡前方 | Limbo, Ori |
| **路径分支** | 上路安全但慢，下路危险但有捷径/奖励 | Super Mario Bros (水管分支) |
| **节奏强制** | 自动滚屏/传送带强制移动速度，玩家只能跳 | Mario 自动滚屏关卡 |
| **环境交互** | 开关触发平台升降/门开关，需要折返或合作 | Trine, Braid |
| **垂直下落** | 从高处向下坠落，躲避两侧伸出的障碍 | Downwell, Ice Climber (反向) |
| **弹射组合** | 连续弹跳平台/弹簧形成弹射链 | Sonic, Ori |
| **敌人利用** | 必须踩踏敌人作为跳板才能到达高处 | Mario (连踩龟壳) |
| **对称/镜像** | 左右对称的关卡，两侧难度不同 | VVVVVV |
| **渐进揭示** | 同一机制在关卡内从简到难出现 3 次 | GMTK 四步法 |

## 维护指南

**新增模板时**（3 步，30 秒）：

1. 在【完整登记表】追加一行，填写博弈组合摘要
2. 在【快速复制区】的代码块中追加对应的一行摘要
3. 如果该模板覆盖了【灵感池】中的某个维度，在该维度后标注 `[已覆盖: T0x]`
