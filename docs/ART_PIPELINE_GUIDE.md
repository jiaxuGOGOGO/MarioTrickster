# MarioTrickster 美术管线白话速查表

> **前端极简自然语言模式已启用。** 从现在起，你不用填表，不用背菜单，不用写长口令。你只要用大白话告诉我“你刚炼完什么 LoRA”或者“你现在要做什么资产”，剩下的查表、补全、配方组合、节点分发、记忆落库与提交推送，全部由我在后台静默完成。

本文件故意只保留两件事，因为对你真正有用的也只剩两件事。第一件事，是你如何用一句白话让我登记新 LoRA；第二件事，是你如何用一句白话让我直接下发 ComfyUI 量产图纸。除非缺了会让任务根本无法执行的致命条件，否则我不追问，不让你做选择题，也不让你补机械表单。

| 你现在要做什么 | 你只要这样说 | 我会在后台自动做什么 |
|---|---|---|
| **登记新 LoRA** | `我炼完 LoRA 了，文件名是 A，触发词是 B。帮我登记。` | 把 LoRA 写进 `prompts/PROMPT_RECIPES.md`，自动补齐默认继承范围、状态与备注；若甜区权重或专属去污词尚未实测，就先记 `待验证`；随后同步 `SESSION_TRACKER.md`，提交并推送仓库。 |
| **下发量产图纸** | `做一组地刺的静态图。` | 自动识别资产类型，静默调用当前默认 LoRA、实体蓝图、知识抽屉和节点策略，直接输出可复制到 ComfyUI 的四区工单：`LoRA / Prompt / ControlNet / KSampler`。 |

## 1. 让我登记新 LoRA

你以后只要像平时说话一样告诉我就行，例如下面这句就是合法输入。

> 我炼完 LoRA 了，文件名是 `MarioTrickster_Style_epoch_10.safetensors`，触发词是 `trickster_style`。帮我登记进项目记忆库并推上仓库。

我收到后会默认把它当成项目风格资产来处理，而不是要求你再填一堆字段。只要文件名和触发词齐了，我就会自动把这张风格卡登记入库。若你还没试出最佳权重或专属负面词，我会先把这些字段写成 **`待验证`**，状态标成 **`验证中`**，不会因为少了非致命信息就卡住你。

## 2. 让我直接下发量产图纸

你以后只要直接说你要做什么资产就行，例如下面这句就是合法输入。

> 做一组地刺的静态图。

我收到后会默认执行整套后台补全：先查当前可用的 LoRA 资产，再查实体蓝图，然后按任务需要静默调用 **解剖 / 透视 / 动画 / 光影 / 参数** 对应知识抽屉，最后按 **LoRA / Prompt / ControlNet / KSampler** 四区物理隔离下发工单。你不需要先判断自己该走哪个菜单，也不需要先把镜头、材质、动作、ControlNet 类型一次性报全。

| 后台默认铁律 | 实际执行方式 |
|---|---|
| **自然语言优先** | 禁止把用户当前轮次重新拖回表单、填空题或多分支口令。 |
| **100% 上下文补全** | 先读 `prompts/PROMPT_RECIPES.md`，自动继承 LoRA 名录、风格合同、实体蓝图与技法抽屉。 |
| **双轨架构不变** | 上半部技法库负责 How，下半部实体库负责 What；出图必须是“实体蓝图 × 技法抽屉”的十字交叉。 |
| **四区物理隔离** | 风格调用、最终提示词、结构约束、采样参数必须分区输出，绝不揉成一坨长 Prompt。 |
| **冲突熔断优先级** | 永远遵守 `物理防滑步 > 空间透视正确 > 光影材质表现`。 |
| **非必要绝不追问** | 只有在缺少致命条件时，才允许用一句话反问。 |

如果你懒得组织语言，就记住这两句白话就够了。一句是“**我炼完 LoRA 了，文件名 A，触发词 B，帮我登记。**”另一句是“**做一组某某资产。**”从现在开始，这就是 MarioTrickster 的唯一前端入口。
