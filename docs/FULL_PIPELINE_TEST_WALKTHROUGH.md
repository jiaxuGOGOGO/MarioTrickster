# 素材导入全流程测试手册（一问一答版）

> 本文档以用户视角，模拟你从"刚下载好一张商业素材"到"关卡里出现带 Shader 效果的可用物体"的完整操作。
> 每一步都是：**你会遇到什么 → 你该做什么 → 你会看到什么结果**。

---

## 前置准备（只做一次）

### Q：我需要提前装什么？

**A**：
1. `git pull` 拉取最新代码
2. 打开 Unity 项目，等编译完成（底部进度条走完，无报错）
3. 确认菜单栏出现 `MarioTrickster` 菜单（里面有 `AI Smart Slicer` 和 `Asset Import Pipeline`）

### Q：API Key 怎么设置？

**A**：
1. 菜单 `MarioTrickster → AI Smart Slicer`
2. 点右侧的"设置"按钮
3. 粘贴你的 OpenAI API Key（sk-开头那个），点确定
4. **只需要做这一次**，以后打开自动记住

---

## 第一阶段：拿到素材

### Q：我从 itch.io / Unity Asset Store / 其他网站买了一个素材包，下载下来是什么样的？

**A**：通常是一个 zip，解压后里面有若干 PNG 图片。常见两种情况：

| 情况 | 长什么样 | 例子 |
|------|----------|------|
| 合集图 | 一张大图里塞了很多不同物体 | `tileset_complete.png`（地形+道具+角色全在一张图里） |
| 动画条带 | 一行/多行排列的同一角色不同帧 | `hero_run_8frames.png`（8帧跑步动画排成一排） |
| 单独文件 | 每个物体已经是独立 PNG | `tree_01.png`、`spike.png`（最省事） |

### Q：如果已经是单独文件了呢？

**A**：直接跳到「第三阶段」，把文件拖进 Unity 即可。

---

## 第二阶段：AI 智能裁切（处理合集图）

### Q：我有一张大合集图，怎么开始？

**A**：
1. 先把这张大图拖进 Unity 项目的**任意位置**（比如 `Assets/Art/Raw/`，临时放哪都行）
2. 菜单 `MarioTrickster → AI Smart Slicer`（或按 `Ctrl+Shift+A`）

### Q：窗口打开了，然后呢？

**A**：
1. 把你刚才拖进来的那张大图，从 Project 窗口**拖到**窗口里的「素材图片」槽位
2. 你会看到它显示图片尺寸（比如 `2048 × 1024`）

### Q：要不要勾"去除背景色"？

**A**：
- 如果你的素材背景是**纯色**（粉色、绿色、白色等），勾上，然后用颜色选择器点一下那个背景色
- 如果背景已经是**透明**的，不用勾

### Q：然后点哪个按钮？

**A**：点那个大按钮 **「🔍 AI 分析」**

### Q：要等多久？

**A**：通常 3~10 秒。窗口会显示"正在调用 AI 视觉模型分析..."

### Q：分析完了，我看到了什么？

**A**：窗口里会列出 AI 识别到的所有物体，每个都有：
- ✅ 勾选框（默认全选）
- 🎬 或 🖼️ 图标（动画组 or 静态物体）
- 名字（如 `hero_walk_cycle`）
- 帧数（如 `8帧, 8×1`）
- 描述（如 `8-frame walk cycle of the main character`）
- 区域坐标

### Q：如果 AI 判断错了怎么办？

**A**：
- 某个物体你不想要 → 取消它的勾选
- AI 把两个物体合成了一个 → 先导出这个，后面在 Unity 里手动再切一刀（极少发生）
- 整体判断离谱 → 关掉窗口，改用传统模式（`sprite_sheet_slicer.py --cols 8 --rows 4`）

### Q：确认没问题后点哪里？

**A**：点底部大按钮 **「✂️ 裁切选中的 X 个物体并导入项目」**

### Q：裁切完了会发生什么？

**A**：
1. 切好的图片自动保存到 `Assets/Art/Imported/` 目录
2. **Asset Import Pipeline 窗口自动弹出**（就是下一步要用的工具）
3. Console 里会打印每个文件的保存路径

---

## 第三阶段：导入为 Object

### Q：Asset Import Pipeline 窗口弹出来了（或者我手动从菜单打开），然后呢？

**A**：
1. 把你要导入的图片（刚裁切好的，或者本来就是单独文件的）**拖到**窗口里的「Sprites」槽位
2. 可以一次拖多个

### Q：「物理类型」选什么？

**A**：根据这个物体在游戏里是什么：

| 物体是什么 | 选什么 |
|-----------|--------|
| 玩家角色、敌人 | Character |
| 地面、平台、墙壁 | Environment |
| 地刺、锯片、岩浆 | Hazard |
| 粒子、光效、装饰动画 | VFX |
| 金币、宝箱、道具 | Prop |

### Q：「切片模式」选什么？

**A**：

| 你的图是什么 | 选什么 |
|-------------|--------|
| 动画条带（AI 裁切出来的 `*_strip_8f.png`） | 手动 → 填列数和行数 |
| 单个静态物体 | 单帧 |
| 不确定 | 自动（工具会用连通域检测） |

### Q：然后点哪里？

**A**：点 **「一键导入并生成 Object」**

### Q：完成后我能看到什么？

**A**：
1. **Scene 视图里**出现了一个新的 GameObject（带 SpriteRenderer，已经显示你的图片）
2. **Project 窗口** `Assets/Prefabs/Imported/` 下出现了对应的 Prefab
3. 这个 Object 已经自动配好了：
   - Root + Visual 分层结构（符合项目架构）
   - BoxCollider2D（尺寸自动匹配）
   - 正确的 Pivot 和 Sorting Layer

---

## 第四阶段：选 Shader 效果

### Q：Object 已经在场景里了，怎么加效果？

**A**：
1. 在 Scene 或 Hierarchy 里**选中**那个 Object
2. 菜单 `MarioTrickster → SEF Quick Apply`（或按 `Ctrl+Shift+Q`）

### Q：窗口里有哪些效果可以选？

**A**：

| 预设名 | 效果 | 适合什么 |
|--------|------|----------|
| 受击闪白 | 被打时整体闪白 | 角色、敌人 |
| 描边 | 黑色轮廓线 | 所有前景物体 |
| 危险描边 | 红色脉冲描边 | 陷阱、危险物 |
| 溶解死亡 | 像素溶解消失 | 敌人死亡 |
| 幽灵剪影 | 半透明剪影 | 无敌状态、幽灵 |
| 冰冻 | 蓝色冰冻覆盖 | 冰冻状态 |
| 像素化 | 低分辨率像素风 | 传送、变身 |
| 投影 | 脚底阴影 | 所有地面物体 |
| 全套战斗 | 上面多个组合 | 核心战斗角色 |

### Q：我点了一个预设后会发生什么？

**A**：
1. **效果立即生效** — Scene 视图里你能直接看到效果（比如描边出现了）
2. **Prefab 自动更新** — 对应的 Prefab 蓝图自动保存了这个效果配置
3. 不满意？直接点另一个预设覆盖，或者手动在 Inspector 里微调参数

### Q：选完效果就结束了？

**A**：是的。此时你的 Prefab 蓝图已经是"**带效果的完成品**"了。

---

## 第五阶段：落地到关卡

### Q：怎么把这个物体放到关卡里？

**A**：两种方式：

**方式 A — 手动拖放**：
1. 打开你的关卡场景
2. 从 Project 窗口找到 `Assets/Prefabs/Imported/` 下的 Prefab
3. 直接拖到 Scene 视图里，摆到你想要的位置

**方式 B — 注册到关卡生成系统**（如果你用 ASCII 模板建关卡）：
1. 把 Prefab 拖到对应的 `ThemePalette` ScriptableObject 的槽位里
2. 之后 `Build From Text` 生成关卡时会自动使用这个素材

### Q：以后换素材怎么办？

**A**：直接替换 Prefab 里的 Sprite 就行，所有引用这个 Prefab 的关卡自动更新。

---

## 完整流程速查（一句话版）

```
下载素材 → AI Smart Slicer 裁切 → Asset Import Pipeline 生成 Object 
→ SEF Quick Apply 选效果 → 拖到关卡里用
```

**总操作次数**：5 次点击 + 1 次拖放

## 常见问题

### Q：编译报错了怎么办？

**A**：确认你已经 `git pull` 到最新。如果是 `HttpClient` 相关报错，检查项目的 API Compatibility Level 是否设为 `.NET Standard 2.1` 或 `.NET Framework`（Edit → Project Settings → Player → Api Compatibility Level）。

### Q：AI 分析没反应 / 超时？

**A**：
1. 检查网络是否能访问 OpenAI API
2. 检查 API Key 是否正确（窗口里会显示 `sk-...最后6位`）
3. 图片太大（>4096px）时 AI 会慢一些，正常等待即可

### Q：裁切出来的位置不对？

**A**：AI 的判断基于视觉理解，偶尔会有几像素偏差。你可以：
- 在 Unity 里用 Sprite Editor 微调
- 或者对那张图改用传统网格模式（`--cols --rows`）

### Q：我能不能跳过 AI 裁切，直接用 Asset Import Pipeline？

**A**：完全可以。如果你的素材已经是单独文件，直接拖进 `Assets/Art/Imported/`，导入窗口自动弹出。

### Q：效果选错了能改吗？

**A**：随时改。选中物体，重新打开 SEF Quick Apply 选另一个预设就行，会覆盖之前的。
